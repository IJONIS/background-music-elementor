Add customizable background music to your Elementor-powered WordPress website with an elegant player interface.

== Description ==

Background Music for Elementor allows you to easily add background music to your website. The plugin adds a new widget to Elementor that enables you to upload and play audio files with a beautiful and customizable player interface.

### Key Features

* **Easy Integration**: Seamlessly adds to your Elementor editor
* **Customizable Player**: Control the look and position of your audio player
* **Audio Settings**: Options for looping and remembering playback position
* **Position Control**: Place the player anywhere on your website
* **Styling Options**: Customize colors, size, and more
* **Volume Control**: Optional volume slider for user control
* **Mobile Friendly**: Responsive design works on all devices

### How to Use

1. Install and activate the plugin
2. Edit any page with Elementor
3. Find the "Background Music" widget in the widget panel
4. Drag and drop it onto your page
5. Upload your audio file and configure settings
6. Save and preview

### Requirements

* WordPress 5.0 or higher
* Elementor 3.0.0 or higher
* PHP 7.0 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/background-music-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the widget in Elementor by searching for "Background Music"

== Frequently Asked Questions ==

= Can I use multiple background music players on the same page? =

Yes, you can add multiple instances of the Background Music widget to your page.

= Will the music automatically play when someone visits my site? =

No, due to browser autoplay policies, users will need to click the play button to start the music.

= Can I customize the player appearance? =

Yes, the plugin offers extensive customization options including colors, size, border, and position.

= Does this work on mobile devices? =

Yes, the player is fully responsive and works on all devices.

= How can I make the player sticky so it stays on screen when scrolling? =

Select "Fixed (Stays in Viewport)" as the Position Type in the widget settings.
